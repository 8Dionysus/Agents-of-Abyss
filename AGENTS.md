# AGENTS.md

## Project identity

Agents of Abyss (AoA) is an evolving ecosystem of AI agents, reusable techniques, and modular infrastructure.

This repository is currently a high-level canonical statement of AoA, not yet the full implementation home.

## Ecosystem map

- `Agents-of-Abyss` = agent ecosystem, workflows, techniques, infrastructure direction
- `Tree-of-Sophia` = living knowledge architecture that AoA helps build and maintain
- `abyss-stack` = modular local/hybrid AI infrastructure backbone
- `aoa-techniques` = reusable public techniques derived from real projects

## Working assumptions

When editing or extending content related to AoA:
- preserve the distinction between agent operations and knowledge architecture
- prefer reusable patterns over one-off hacks
- prefer modularity over monolithic fusion
- maintain source-of-truth discipline
- keep changes reviewable and well-scoped

## Current repository role

This repository currently serves as:
- public landing page
- canonical high-level project identity
- stable reference point for adjacent repositories and coding agents

## If you need implementation details

Check adjacent repositories first:
- `aoa-techniques`
