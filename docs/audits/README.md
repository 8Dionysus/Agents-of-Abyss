This directory holds audit evidence, cleanup candidates, and review artifacts that should remain public and inspectable without standing in the repository root as civic law.

## Rule

An audit artifact is not authority by itself. It becomes actionable only when:

1. a surviving canonical home is named,
2. the owner boundary is clear,
3. the move preserves provenance,
4. validators or manual checks are listed,
5. the change reduces drift rather than hiding it.
