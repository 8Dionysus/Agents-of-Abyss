# Canary Watch

Canary probes test the released version against behavioral promises. They are small, repeated, evidence-bearing checks.

Probe families:

- scope fidelity
- receipt fidelity
- escalation fidelity
- false closure risk
- output variance
- regression sentinel
- handoff packet integrity
- rollback reachability

Canary failure blocks ring promotion unless an authorized evaluator marks it non-material.
