# Agon Contradiction Law Model

A contradiction becomes material when it is relevant to closure, evidence, trace, scope or canon risk.

Allowed pre-protocol statuses are `open`, `localized`, `resolved`, `deferred_with_cost`, `bifurcated` and `invalidated`.

The key rule is simple: `open` and material blocks lawful closure.

Wave X reserves a contradiction entry shape but does not create durable ledger storage. Durable memory belongs downstream, and only future protocol waves may request memory writes.
