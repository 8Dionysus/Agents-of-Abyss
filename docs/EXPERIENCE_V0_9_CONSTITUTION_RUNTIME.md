# Experience v0.9: Constitution Runtime

v0.9 turns the polis-governance layer into a runtime contour. It does not create a new sovereign. It creates queues, jobs, locks, sealed records, replay chains, and enforcement hooks so that governance decisions can run without becoming invisible doctrine.

The central rule is simple: law may execute, but execution must remain reviewable.

The runtime is allowed to route a case, schedule a council, enforce a live stay order, expire an appeal window, index a sealed precedent, and produce audit reports. It is not allowed to decide truth, certify itself, or launder authority through automation.
