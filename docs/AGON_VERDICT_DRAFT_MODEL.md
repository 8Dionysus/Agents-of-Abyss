# Agon Verdict Draft Model

A verdict draft is a pre-runtime artifact that records what a future judge would need to say after an arena session. It is not a live decision and cannot grant closure, mutate rank, or write memory.

Terminal outcomes reserved by this wave:

```text
resolved
revised_and_resolved
bifurcated
deferred_with_cost
failed_honorably
invalidated
quarantined
```

The verdict draft is the first place where the future arena refuses theatrical closure. It demands that every ending name its cost, remaining tension, and owner handoffs.
