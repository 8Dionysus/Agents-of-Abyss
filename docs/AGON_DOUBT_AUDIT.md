# Agon Doubt Audit

This document provides the first audit frame for placing the existing AoA federation under Agon scrutiny.

It is not a final verdict table. It is the review instrument.

## Audit questions

Every repository or surface under review must answer:

1. What does it currently claim to own?
2. Which of those claims become suspicious under Agon?
3. Which parts strengthen lawful conflict, witnessability, actor formation, or source ownership?
4. Which parts risk authority drift, hidden autonomy, premature runtime, false canonization, or decorative combat language?
5. Should the surface survive, recharter, defer, prune, or quarantine?
6. Which owner repository must receive the next action?
