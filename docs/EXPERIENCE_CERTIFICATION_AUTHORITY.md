# Experience Certification Authority


Roles:

- Codex: capture, propose, run checks, assemble evidence.
- Assistant agent: emit service incident signal and request review.
- Evaluator: issue bounded verdict and regression verdict.
- Operator/reviewer: certify or deny release.
- Memo: store revision ledger and post-release retention memory.
- Routing: dispatch owner fixes and gate release surfaces.

Forbidden: Codex self-certification, assistant self-certification, stats-as-judge, memo-as-proof.
