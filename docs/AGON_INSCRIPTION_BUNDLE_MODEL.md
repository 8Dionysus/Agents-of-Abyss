# Agon Inscription Bundle Candidate Model

An inscription bundle candidate groups verdict draft, delta candidates, scar requests, retention requests, stats hints and owner routes.

The bundle is not durable inscription. It does not write to `aoa-memo`, does not promote to `aoa-kag`, and does not approach `Tree-of-Sophia`.

It is a sealed satchel at the arena door, not a tablet in the archive.
