# Agon State Packet Model

Wave IX gives Agon a state packet grammar without creating live sessions.

A state packet is a reviewable, typed, pre-protocol carrier. It may describe a draft opening, a charter acceptance, a sealed commit, a reveal view, a move declaration, an evidence reference, a contradiction notice, a summon intent, a witness request, a revision statement, an adjudication request or an inscription candidate.

It does not execute the move it names.

The packet layer exists so that future arena sessions can stop being theatrical conversation. Contestants must be able to commit before reveal, reveal against a prior commitment, declare lawful moves, and make revision explicit after pressure.

The packet layer does not own proof, memory, routing, stats, skills, techniques or runtime. It only defines what shape a future packet must have before those owners receive it.
