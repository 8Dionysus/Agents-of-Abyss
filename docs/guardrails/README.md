# Docs Guardrails

This district owns docs-local guardrail law: cleanup classification, link and shape hygiene, generated freshness, and AGENTS mesh coverage.

## District law

Guardrails keep route surfaces checkable. They do not author center doctrine, mechanic truth, generated meaning, or sibling-repo authority.

## Must not claim

Guardrails do not decide what the center means. They check whether routes, links, generated mirrors, and local agent gates remain coherent.

Do not place mechanic-owned history here. Mechanic receipts belong under `mechanics/<slug>/legacy/`; this district may only point to those owner routes.

## Promotion path

A guardrail change becomes current only when the source config or protocol, generated mirror, validators, tests, and docs links move together.
