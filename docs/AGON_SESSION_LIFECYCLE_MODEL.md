# Agon Session Lifecycle Model

Wave VIII models the lifecycle of future arena sessions without executing it.

```text
gate
charter
seat_assignment
sealed_commit_slot
reveal_slot
engagement_window
pressure_window
revision_window
adjudication_slot
inscription_slot
retention_slot
closeout
```

The sealed commit and reveal slots are reserved for Wave IX. Adjudication, inscription, and retention are reserved for later waves. Wave VIII only establishes where those organs must sit.
