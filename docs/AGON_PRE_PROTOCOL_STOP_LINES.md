# Agon Pre-Protocol Stop-Lines

Wave IV is still pre-protocol.

Do not let owner binding smuggle in:

```text
arena runtime
packet transport
sealed commit
contradiction ledger storage
verdict law
scar writing
retention scheduling
rank mutation
Tree-of-Sophia promotion
```

A binding that starts executing is no longer a binding.

A binding that starts judging is no longer a binding.

A binding that starts storing scars is no longer a binding.

A binding that starts routing live actors is no longer a binding.

A binding that starts claiming canon is a fire in the archive.

The correct motion is:

```text
lawful move -> owner binding -> owner candidate -> later reviewed owner landing
```

Nothing jumps the queue.
