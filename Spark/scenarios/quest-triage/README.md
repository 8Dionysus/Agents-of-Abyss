# Spark Scenario: quest-triage

Use `quest-triage` to route durable tails into quest candidates without turning
private scratch work into public obligations.

## Scope

One obligation lane or one source cluster.

## Done Signal

Quest candidates are routed to the right lane or handoff.

## Stop-line

Do not turn private scratch work into public quest obligations.

## Handoff Route

Write a handoff when a quest needs owner decision, downstream acceptance, or
campaign design.

## Validation

Run Questbook source and lifecycle validators when quest files change.
