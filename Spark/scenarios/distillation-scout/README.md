# Spark Scenario: distillation-scout

Use `distillation-scout` to map legacy or raw material before a deeper
distillation session.

## Scope

Read legacy/raw sources, name active homes, and identify unresolved owner
questions.

## Done Signal

Legacy sources are mapped to likely active homes and risks.

## Stop-line

Do not perform deep semantic distillation inside scout mode.

## Handoff Route

Write a handoff when the material needs synthesis, active doctrine changes, or
owner-local acceptance.

## Validation

Run shape or link checks when the scout updates an index. Otherwise report a
manual provenance pass.
