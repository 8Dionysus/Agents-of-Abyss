# Recurrence Roadmap

## Current Contour

Recurrence now has a center package that can carry real work without becoming
ambient continuity:

- `DIRECTION.md` names the present route: honest return, bounded re-entry, and
  owner-local follow-through.
- `PARTS.md` splits the mechanic into active parts rather than long historical
  doctrine.
- `OWNER_MAP.md` keeps stronger-owner routes visible.
- `OWNER_REQUESTS.md` holds ready-to-carry packets for sibling repositories.
- `PROVENANCE.md` preserves the deeper source trail without loading active
  parts with inventories.

The current job is not to make recurrence automatic. It is to make every return
claim reviewable, owner-routed, and small enough to use in live work.

## Next Work

- Keep owner-local recurrence packages light when sibling repositories receive
  their own `mechanics/recurrence/` homes.
- Use `OWNER_REQUESTS.md` whenever a center recurrence part produces work that
  belongs in `aoa-sdk`, `aoa-routing`, `aoa-memo`, `aoa-agents`,
  `aoa-playbooks`, `aoa-evals`, `aoa-stats`, `aoa-kag`, `abyss-stack`, or
  `ATM10-Agent`.
- Add machine-readable recurrence metadata only after the text part map proves
  stable across at least one owner-local landing.
- Keep `LANDING_LOG.md` indexed and readable as the package grows.

## When The Time Comes

Add to this block only when the work is not needed now, but the future trigger
is concrete enough that the next agent should not rediscover it from scratch.

- When `aoa-sdk` accepts recurrence control-plane work, add a manifest-to-part
  map that links carry packets, graph closure, projection, and reviewed handoff
  back to `control-plane-carry`.
- When `aoa-agents` accepts recursor contracts, add recursor activation law that
  keeps no-spawn boundaries explicit before any role can recurse.
- When `abyss-stack` accepts runtime-return work, add a product-runtime route
  table that separates runtime receipts, restart semantics, and operator
  recovery from center recurrence law.
- When `aoa-memo` accepts recurrence recall objects, add a memory handoff table
  that keeps recall anchors separate from ambient memory claims.

## Out Of Scope

- Hidden memory sovereignty.
- Runtime self-healing claims.
- Automatic recursor spawn.
- Product-runtime activation from the center package.
- Agon-specific recurrence authority outside `mechanics/agon/`.
