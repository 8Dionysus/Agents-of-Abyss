# Memory Recall Contract

## Center Owns

The center owns the stop-line that memory supports return through explicit
anchors and traces without becoming route authority.

## Must Not Claim

- hidden memory sovereignty
- return_memory canon
- role rights
- dispatch authority

## Allowed Outputs

- memo support request
- recall route
- state capsule route
- decision or audit-event route
- provenance thread route

## Required Before Output

- one anchor or trace class
- one owner-local memo route
- no new memory family unless `aoa-memo` accepts it
