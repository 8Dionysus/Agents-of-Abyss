# Runtime Return Contract

## Center Owns

The center owns the stop-line that runtime return must stay downstream of
anchors, owner gates, and proof.

## Must Not Claim

- runtime self-healing
- ambient autonomy
- direct runtime resume
- federation-wide runtime policy from product-local evidence

## Allowed Outputs

- runtime return request
- context rebuild boundary
- return-event evidence route
- safe-stop route
- runtime-owner handoff

## Required Before Output

- explicit anchor or safe-stop reason
- owner-local runtime policy target
- proof or evidence route for public claims
