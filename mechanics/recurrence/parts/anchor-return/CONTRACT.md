# Anchor Return Contract

## Center Owns

The center owns the law that return must target a valid anchor rather than
ambient continuity.

## Must Not Claim

- blind retry
- hidden context inflation
- ambient continuity
- owner-local implementation truth

## Allowed Outputs

- return reason
- anchor reference
- re-entry note
- safe stop
- owner handoff

## Required Before Output

- one lost axis or explicit safe-stop reason
- one inspectable anchor or explicit no-anchor finding
- one stronger owner route
