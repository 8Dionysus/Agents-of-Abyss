# Scenario Choreography Contract

## Center Owns

The center owns the rule that recurring return method should become
playbook-owned once it becomes scenario choreography.

## Must Not Claim

- scheduler authority
- automatic orchestration
- runtime execution
- playbook truth before owner landing

## Allowed Outputs

- playbook request
- return-trigger language
- anchor-class route
- re-entry-mode route
- fallback or safe-stop route

## Required Before Output

- repeated scenario shape
- explicit owner route
- fallback or stop posture
