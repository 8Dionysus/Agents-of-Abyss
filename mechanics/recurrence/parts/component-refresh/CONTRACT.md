# Component Refresh Contract

## Center Owns

The center owns the law that drifting components refresh through owner-owned
receipts rather than hidden self-maintenance.

## Must Not Claim

- automatic regeneration
- stats-owned refresh truth
- scheduler authority
- runtime self-healing

## Allowed Outputs

- component refresh request
- owner route
- drift hint boundary
- reviewed decision route
- proof or receipt request

## Required Before Output

- one named component
- one owner repo
- one rollback anchor or explicit gap
- one validation or proof route
