# Raw Intake

Use this part to decide whether material may enter a distillation route.

## Use When

- A raw packet, long note, reviewed checkpoint, donor artifact, runtime pack, or
  witness-facing surface needs conversion into active form.
- The review state, source owner, or stop-line is unclear enough to name before
  extraction.

## Do Not Use When

- The material is already an active part and only needs normal editing.
- The task is raw capture, transcript export, or source preservation without an
  active extraction question.

## Route Check

Name the source, owner, review state, intended active use, and first stop-line
before extracting anything.

## Active Outputs

- Intake decision.
- Source-owner note.
- Review-state note.
- First active part route.

## Next Route

Use `raw-preservation` when the source must be kept, `active-extraction` when a
function survives, and `candidate-handoff` when the result belongs to another
owner.
