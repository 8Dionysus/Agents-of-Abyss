# Raw Preservation

Use this part to preserve raw material without letting it become active law.

## Use When

- Source material remains valuable as evidence, history, or donor input.
- A packet should be retained but not placed in the active route.

## Do Not Use When

- The material should be deleted under a separate retention or pruning law.
- The active function has not yet been extracted.

## Route Check

Name what is preserved, why it remains useful, where it is stored, and how
future agents reach it without defaulting to it.

## Active Outputs

- Preservation route.
- Legacy or source-owner pointer.
- Stop-line against active-law inflation.

## Next Route

Use `provenance-bridge` for controlled source references and `noise-pruning`
for material that should leave active surfaces.
