# Agon Retention Check Model

Retention checks ask whether a future actor still behaves differently after a prior rupture, trial candidate, scar request, or delta candidate.

Allowed check classes include lesson retention, trace integrity retention, closure discipline retention, summon cost retention, assistant boundary retention, prediction accuracy retention, and contradiction handling retention.

A retention request is a reviewable candidate. It is not hidden scheduling, not runtime execution, and not a durable memory write.
