# Agon Sealed Commit Model

The sealed commit is the first anti-mimicry organ of the arena.

A contestant must be able to bind a position before seeing the full revealed posture of the other side. The sealed commit packet carries a public summary and a commitment hash over hidden fields such as thesis, confidence, beliefs at risk, revision conditions, predicted other, affect snapshot, self-image claim and stake acceptance.

Wave IX does not define a live cryptographic service. It only defines the packet surface and review obligations.

Future runtime may compute and verify hashes. This seed only says that a future arena must have a place for that discipline.
