# Agon Recurrence Stop-Lines

The adapter inherits the Agon stop-lines and adds recurrence-specific ones.

## Hard stop-lines

- No arena session creation.
- No sealed commit creation.
- No live move execution.
- No verdict authority.
- No scar write.
- No retention scheduling.
- No rank mutation.
- No Tree-of-Sophia promotion.
- No hidden scheduler.
- No automatic owner mutation.
- No assistant contestant drift.
- No source meaning rewrite.

## Meaning

A recurrence observation can say:

```text
this pressure repeated
this owner should review
this dossier should be prepared
this surface may have drifted
```

It cannot say:

```text
this agent won
this scar is written
this rank changed
this branch is canon
```
