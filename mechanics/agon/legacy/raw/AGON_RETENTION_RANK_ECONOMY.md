# Agon Retention and Rank Economy

Wave XIV defines the first economy of time-tested consequence for Agon.

A scar request is not enough. A delta candidate is not enough. A trial outcome candidate is not enough. Agon needs a way to ask whether the lesson survived re-entry.

This wave defines retention check requests, retention result candidates, rank standing models, and jurisdiction mutation candidates. It does not mutate rank, trust, scar memory, closure authority, summon authority, KAG status, or Tree of Sophia status.

The core law is simple: no standing is earned by a single blaze. Standing must pass through witnessable recurrence, owner review, and retention pressure.
