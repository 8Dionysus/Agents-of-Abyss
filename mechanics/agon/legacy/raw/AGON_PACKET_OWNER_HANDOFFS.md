# Agon Packet Owner Handoffs

`Agents-of-Abyss` owns the packet law.

Other owners keep their truth:

- `aoa-agents` owns actor formation and eligibility surfaces.
- `aoa-routing` owns gate hints.
- `aoa-playbooks` owns trial choreography.
- `aoa-techniques` owns reusable practice candidates.
- `aoa-skills` owns bounded workflow candidates.
- `aoa-evals` owns future proof and legality checks.
- `aoa-memo` owns future memory candidate intake and durable memory later.
- `aoa-stats` owns derived observability.
- `aoa-sdk` may host typed helper candidates without becoming runtime authority.
- `abyss-stack` may later host durable runtime, but this wave does not touch it.
