# Gate Routing Contract

## Owner boundary

`Agents-of-Abyss` owns the pre-protocol gate-routing request. `aoa-routing`
owns live route behavior after owner-local landing.

## Allowed outputs

- gate request
- route handoff
- route stop-line
- proof route hint

## Stop-lines

- no arena activation
- no verdict authority
- no rank mutation
- no runtime dispatch
- no ToS promotion
