# Trial Handoff Contract

## Owner boundary

`Agents-of-Abyss` owns the pre-protocol trial-playbook request. `aoa-playbooks`
owns recurring trial choreography only after owner-local landing.

## Allowed outputs

- trial-playbook request
- setup and closure hints
- expected evidence route
- stop-line for rehearsal

## Stop-lines

- rehearsal is not live arena execution
- no verdict, scar, retention, rank, runtime, or ToS authority
- no assistant contestant authority
