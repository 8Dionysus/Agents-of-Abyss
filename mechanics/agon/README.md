# Agon Mechanic

Agon is the center mechanic for bounded pressure.
It turns conflict, contradiction, tests of survival, and adversarial reading
into reviewable moves without making the center a live arena.

## Owner Boundary

`Agents-of-Abyss` owns Agon vocabulary, stop-lines, center doctrine, and owner
handoff expectations.
Sibling repositories own their own practice, workflows, proof, memory, stats,
routing, actor contracts, runtime behavior, and ToS meaning.

## Must Not Claim

- Agon is live arena execution in this repository.
- Agon grants assistant contestant authority.
- Agon mutates rank, memory, verdict, scars, or runtime state here.
- Agon can write ToS canon directly.
