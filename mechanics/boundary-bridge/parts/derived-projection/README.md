# Derived Projection

Use this part when source-owned material becomes KAG/export/graph-ready or
retrieval-ready without becoming source authority.

## Function

Keep derived surfaces visibly downstream of source owners.

## Inputs

- source refs
- projection owner
- derived shape
- proof route if public

## Outputs

- derived projection boundary
- source-link requirements
- non-source-authority note

## Stop-lines

- no source canon inside projection
- no projection as proof
- no graph convenience over source authority

## Validation

Use the validation lane in [../../AGENTS.md](../../AGENTS.md#validation).
