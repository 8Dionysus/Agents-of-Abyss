# ToS Support

Use this part when AoA supports Tree-of-Sophia without authoring ToS meaning.

## Function

Route ToS-adjacent support, counterpart, witness, compost, growth, template,
lineage, or soil-prep work while keeping ToS source authority in
`Tree-of-Sophia`.

## Inputs

- ToS-owned source or doctrine surface
- AoA support surface
- support mode
- stop-line against AoA-authored ToS meaning

## Outputs

- ToS support route
- ToS owner handoff when needed
- proof or review route where support claims need evidence

## Stop-lines

- no AoA-authored ToS canon
- no source interpretation authority in AoA
- no direct ToS write from bridge support
