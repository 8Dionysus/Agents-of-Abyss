# Counterpart Edge

Use this part when two surfaces can illuminate each other without becoming one
object.

## Function

Carry counterpart vocabulary: `analogy`, `support`, `tension`, and
`calibration`.

## Inputs

- source concept or surface
- operational neighbor
- counterpart mode
- evidence for usefulness
- non-identity note

## Outputs

- counterpart edge
- source refs
- derived projection route when needed

## Stop-lines

- no proof of philosophical identity
- no operational meaning written back into source nodes
- no KAG projection as canon

Use the validation lane in [../../AGENTS.md](../../AGENTS.md#validation).
