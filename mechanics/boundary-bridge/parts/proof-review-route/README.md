# Proof Review Route

Use this part when a bridge claim needs proof, source review, or public support
evidence.

## Function

Separate source authority, owner receipt, eval proof, public support, and
generated companion evidence.

## Inputs

- bridge claim
- owner receipt status
- proof need
- public claim risk

## Outputs

- proof route
- supportable claim boundary
- evidence requirement

## Stop-lines

- no public claim without evidence
- no proof outside proof owner
- no generated companion as authority

## Validation

Use the validation lane in [../../AGENTS.md](../../AGENTS.md#validation).
