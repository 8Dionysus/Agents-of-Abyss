# Compatibility Route

Use this part when SDK, routing, public projection, generated companions, or
adapters need a stable bridge carrier.

## Function

Let compatibility surfaces carry bridge information without becoming meaning
owners.

## Inputs

- compatibility owner
- source owner
- bridge surface
- adapter or route behavior

## Outputs

- compatibility route
- owner split
- no-authority note

## Stop-lines

- no adapter as meaning
- no route as source truth
- no public projection ahead of owner evidence
