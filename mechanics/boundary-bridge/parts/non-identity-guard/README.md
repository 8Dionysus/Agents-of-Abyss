# Non-Identity Guard

Use this part when a bridge could be mistaken for sameness.

## Function

Keep support, analogy, projection, translation, compatibility, and route
language from becoming an identity claim.

## Inputs

- bridge statement
- two bridged surfaces
- possible confusion

## Outputs

- non-identity note
- stronger owner note
- forbidden reading

## Stop-lines

- no "X is Y" unless the source owner says so
- no counterpart edge as proof
- no derived projection as source truth

Use the validation lane in [../../AGENTS.md](../../AGENTS.md#validation).
