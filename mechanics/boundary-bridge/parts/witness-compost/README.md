# Witness Compost

Use this part when witness material may move toward compost without collapsing
memory, proof, runtime, and ToS canon.

## Function

Route a witness-facing artifact into reviewable compost posture while each
owner keeps its lane.

## Inputs

- witness artifact
- memory owner route
- proof route
- ToS compost route if applicable
- runtime deferral if applicable

## Outputs

- witness route
- compost handoff
- proof boundary
- memory boundary

## Stop-lines

- no hidden runtime trace platform
- no memory object as proof
- no compost note as canon without ToS review

## Validation

Use the validation lane in [../../AGENTS.md](../../AGENTS.md#validation).
