# Evidence Ledger

## Use When

Use this part when audit work needs a reviewable list of evidence, missing evidence, freshness, and confidence limits.

## Do Not Use When

Use `aoa-evals` when the work requires proof verdicts, comparisons, or regression claims.

## Route Check

Separate observed evidence, absent evidence, stale evidence, and derived signals.

## Active Outputs

- evidence ledger
- missing-evidence note
- freshness note
- derived-signal note
- proof route hint

## Next Route

Route proof-strength claims to `aoa-evals`; route durable recall or memory writeback to `aoa-memo`.
