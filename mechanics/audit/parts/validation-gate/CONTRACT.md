# Validation Gate Contract

## Owner Boundary

This part owns check-report posture for center audit. It does not own proof, CI policy, or source-local validation law.

## Allowed Outputs

- checks-run note
- skipped-check note
- failure summary
- stale-generated note
- residual risk

## Stop-lines

- Do not turn green local checks into global proof.
- Do not omit skipped checks.
- Do not treat generated freshness as source meaning.

## Validation

Use the validation lane in [mechanics/audit/AGENTS.md](../../AGENTS.md#validation).
