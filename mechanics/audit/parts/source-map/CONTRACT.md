# Source Map Contract

## Owner Boundary

This part owns center-level source orientation for audit work. It does not own the source surface.

## Allowed Outputs

- source path
- owner route
- source-of-truth posture
- generated or legacy posture
- missing owner context

## Stop-lines

- Do not treat proximity as ownership.
- Do not promote generated output into source truth.
- Do not let a legacy receipt override active source surfaces.

Use the validation lane in [mechanics/audit/AGENTS.md](../../AGENTS.md#validation).
