# Audit Event Bridge

## Use When

Use this part when an audit event must connect to checkpoint, questbook, release-support, growth-cycle, distillation, antifragility, or owner-request routes.

## Do Not Use When

Use the crossed mechanic directly when the work already belongs there and needs no audit bridge.

Name the audit event, crossed mechanic, owner boundary, handoff reason, and landing or closure surface.
