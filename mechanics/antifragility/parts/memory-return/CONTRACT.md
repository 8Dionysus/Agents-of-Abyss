# Memory Return Contract

## Center Boundary

AoA may name what lesson should return. `aoa-memo` owns memory object truth.

## Allowed Outputs

- Lesson route.
- Retention reason.
- Forgetting boundary.
- Recall handoff.

## Stop-lines

- Memory is not proof.
- Recall is not live readiness.
- Raw history should stay behind `PROVENANCE.md`.
