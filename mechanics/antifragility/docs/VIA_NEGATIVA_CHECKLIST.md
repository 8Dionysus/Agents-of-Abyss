# VIA_NEGATIVA_CHECKLIST

This checklist is for `Agents-of-Abyss` as the federation center.

## Keep intact

- the constitutional map of layers and ownership boundaries
- short center-level prohibitions that prevent authority drift
- route-map guidance that points outward to real owners

## Merge, move, or remove when found

- center docs that restate repo-local run instructions, support matrices, or
  proof semantics
- repeated ecosystem doctrine split across multiple center docs without a clear
  need
- seed-derived temporary guidance that has already been absorbed into a
  canonical center file

## Questions before adding anything new

1. is this truly a center rule, or should it live in an owner repo?
2. can this be one paragraph in an existing constitutional doc instead of a
   new file?
3. will this age as a stable prohibition, or is it a short-lived wave note?

## Safe exceptions

- short index or glossary updates that improve routing clarity
- small constitutional additions that block a real class of authority drift

## Exit condition

The center stays thin: stronger as a map, lighter as an owner.
