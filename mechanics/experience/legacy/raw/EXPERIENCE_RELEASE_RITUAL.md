# Experience Release Ritual


The ritual is:

1. accept reviewed patch proposal,
2. build release candidate,
3. attach behavior contract delta,
4. run regression pack,
5. run rollback drill,
6. evaluate certification gate,
7. write operator review item,
8. issue or deny certification,
9. publish release note,
10. schedule post-release retention watch.

This ritual keeps assistant growth external, visible and reversible.
