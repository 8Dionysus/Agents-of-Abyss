## Use when

Use when Experience names runtime-adjacent behavior while keeping activation outside the center repository.

- runtime stop-line, authority resolver boundary, case queue handoff, projection boundary, and rollback route.
- Owner handoff language when the center cannot accept or prove the operational slice.
