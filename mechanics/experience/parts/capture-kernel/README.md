# Capture Kernel

Capture Kernel is an active Experience part. It owns the first bounded conversion of lived friction, incidents, candidates, verdict routes, memory gates, and inert projections into reviewable Experience inputs.

## Use when

Use when a work session, release friction, incident, or observed pattern needs to become an Experience object without claiming runtime or memory authority.

## Active outputs

- friction record, candidate packet, owner route, memory gate, and inert projection boundary.
- Owner handoff language when the center cannot accept or prove the operational slice.
- Stop-lines that keep Experience center law from becoming live runtime, memory, proof, or owner-local truth.
