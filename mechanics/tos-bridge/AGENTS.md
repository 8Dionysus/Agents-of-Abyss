# AGENTS.md

This file applies to `mechanics/tos-bridge/`.

## Role

ToS bridge owns AoA support doctrine for Tree-of-Sophia coordination.
It may describe support, witness, compost, soil prep, templates, and lineage
pilot routes.

It does not author ToS meaning or move ToS canon into AoA.

## Validation

Run `python scripts/validate_mechanics_topology.py --mechanic tos-bridge` after
package changes.
