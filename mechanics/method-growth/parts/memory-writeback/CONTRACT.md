# Memory Writeback Contract

## Center Boundary

The center may ask for memory writeback and name retention reasons. It does not
own memory canon or recall contracts.

## Allowed Outputs

- Lesson route.
- Retention reason.
- Recall hint.
- Pruning receipt route.

## Stop-lines

- A lesson is not memory canon.
- A closeout note is not a recall object.
- Raw provenance should not be copied into every active doc.
