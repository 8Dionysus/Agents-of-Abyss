# Quest Store Contract

## Owner Boundary

AoA center owns the common source layout. Lanes and owner repositories preserve
their own obligation meaning and acceptance evidence.

## Allowed Outputs

- lane README route note
- source quest object
- lifecycle state move

## Stop-lines

- Do not create root `AOA-Q-*` aliases.
- Do not create root lifecycle directories such as `quests/ready/`.
- Do not move owner-local task truth into AoA center unless it is a federation
  obligation.
