# Source Contract Contract

## Owner Boundary

AoA center owns the common reviewability contract for Questbook source objects.
Lanes and owner repositories own the obligation meaning, acceptance evidence,
and closure proof.

## Allowed Outputs

- strict YAML source contract check
- strict Markdown source contract marker
- source object reviewability rule

## Stop-lines

- Do not create or retain Markdown quest sources without
  `source_contract: quest_markdown_contract_v1`.
- Do not treat a strict source contract as owner acceptance or proof.
- Do not put private local notes into public quest source files.
- Do not let generated read models replace source quest review.
