# Quest Promotion

Quest Promotion decides whether reviewed repeated work becomes a public
obligation rather than private residue.

## Use When

- A repeated unresolved pressure should survive beyond the current diff.
- The route has enough evidence to be carried publicly.
- The task fits quest lanes better than roadmap prose or private notes.

## Do Not Use When

- The item is scratchpad residue.
- The work already belongs in an owner request, roadmap, or release note.
- There is no reviewed evidence or owner route.

## Route Check

Ask whether the obligation is public, durable, owner-routed, testable, and not
already covered by an existing quest or owner request.

## Active Outputs

- quest promotion request
- lane hint
- owner route
- evidence refs
- closure proof hint
