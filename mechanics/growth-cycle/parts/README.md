# Growth Cycle Parts Index

Active Growth Cycle parts live here. Each part is a stage route, not an
implementation surface.

## Parts

- [Checkpoint Intake](checkpoint-intake/README.md): provisional carry enters the reviewed cycle.
- [Reviewed Closeout Chain](reviewed-closeout-chain/README.md): review gates harvest, progression, repair, quest, memory, and owner claims.
- [Donor Harvest](donor-harvest/README.md): reusable residue becomes owner-routed candidate pressure.
- [Progression Lift](progression-lift/README.md): evidence-backed movement is lifted without a universal score.
- [Route Forks](route-forks/README.md): material next choices are named with costs and stop conditions.
- [Automation Opportunity](automation-opportunity/README.md): automation seeds are noticed without hidden activation.
- [Diagnosis Gate](diagnosis-gate/README.md): diagnosis comes before repair.
- [Repair Cycle](repair-cycle/README.md): repair stays smallest, reversible, and proof-routed.
- [Quest Promotion](quest-promotion/README.md): repeated work becomes public obligation only after review.
- [Owner Followthrough](owner-followthrough/README.md): unresolved pressure leaves as owner requests and handoff packets.
