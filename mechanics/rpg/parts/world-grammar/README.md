# World Grammar

This part owns the active rule for when RPG language should appear at all.

RPG language is useful only when it improves routing, judgment, memory, proof, or consequence. It should stay silent when it would merely rename ordinary work.

## Active Function

- name role, origin, quest, campaign, party, resource, reputation, unlock, and consequence only when they clarify action
- keep game language tied to owner boundaries and proof routes
- prevent decorative game skin from becoming route noise

## Next Route
