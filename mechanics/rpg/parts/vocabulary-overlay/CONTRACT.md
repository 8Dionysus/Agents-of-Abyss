# Vocabulary Overlay Contract

## Center Owns

- canonical RPG key vocabulary
- themed-label separation law
- public-safe overlay schema and example

## Must Not Claim

- that presentation labels replace canonical keys
- runtime state or frontend authority
- owner-local role, skill, quest, proof, or memory truth

## Allowed Outputs

- vocabulary key
- themed label
- generated overlay mirror
- validation result

## Required Before Output

- canonical key remains visible
- themed label is demonstrably derived
- schema, example, and generated mirror stay synchronized when vocabulary changes
- owner-local role, skill, quest, proof, memory, and runtime claims stay outside the overlay
