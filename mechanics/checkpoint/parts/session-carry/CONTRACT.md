# Session Carry Contract

## Center Owns

The center owns the rule that session carry is provisional.

## Must Not Claim

- reviewed candidate identity
- seed identity
- owner acceptance
- memory canon
- proof verdict

## Allowed Outputs

- checkpoint signal
- `cluster_ref`-level hint
- evidence refs
- owner hint
- next review route

## Required Before Output

- one named boundary event
- one evidence ref or explicit reason evidence is unavailable
- one stronger owner hint or an explicit unknown-owner route
