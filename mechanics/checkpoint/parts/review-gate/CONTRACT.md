# Review Gate Contract

## Center Owns

The center owns the stop-line that checkpoint evidence cannot silently promote
itself.

## Must Not Claim

- approval already granted
- proof already passed
- owner accepted the route
- runtime is safe to continue

## Allowed Outputs

- review gate
- approval request
- rollback marker request
- proof route
- safe stop

## Required Before Output

- named risk or boundary
- named reviewer or owner route
- explicit next action after review
