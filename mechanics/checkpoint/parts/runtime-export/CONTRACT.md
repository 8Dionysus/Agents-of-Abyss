# Runtime Export Contract

## Center Owns

The center owns the rule that runtime checkpoint exports remain runtime-owned
until reviewed and routed.

## Must Not Claim

- runtime activation
- runtime safety
- memory canon
- replay integrity
- live automation

## Allowed Outputs

- export boundary
- runtime owner request
- writeback route
- proof route
- safe stop

## Required Before Output

- runtime owner named
- export kind named
- review or proof route named when a claim would survive the run
