# Return Re-entry Contract

## Center Owns

The center owns the rule that return must re-enter from a valid anchor.

## Must Not Claim

- ambient continuity
- hidden memory sovereignty
- direct runtime resume
- route authority

## Allowed Outputs

- return anchor
- re-entry note
- safe stop
- memo recall request
- routing hint request

## Required Before Output

- one valid anchor or explicit safe stop
- one owner route
- one bounded next step
