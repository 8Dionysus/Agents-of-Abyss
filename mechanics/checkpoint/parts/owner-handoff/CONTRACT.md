# Owner Handoff Contract

## Center Owns

The center owns the request packet and stop-line.

## Must Not Claim

- owner acceptance
- owner-local landing
- owner proof
- operational truth

## Allowed Outputs

- request ID
- owner route
- required owner landing
- proof route
- next action

## Required Before Output

- target owner
- center source
- stop-line
- required owner landing
- next action
