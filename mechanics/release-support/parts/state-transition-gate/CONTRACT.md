# State Transition Gate Contract

Every transition claim must name:

- previous state
- target state
- evidence surface
- owner boundary
- public visibility
- rollback or return route

## Required Before Output

- The transition is not only a synonym or formatting change.
- The owner of the target state is explicit.
- Unsupported claims are named as planted, requested, blocked, or future work.

## Stop-lines

- No state transition by label alone.
- No owner acceptance without owner receipt.
- No generated surface as transition authority.
